English

This Privacy Policy explains how My Moments (the “App”) collects, uses, stores, and protects user information. By using the App, you agree to this Privacy Policy.

1. Information We Collect

The App can be used without creating an account. However, the App offers an optional backup and restore feature that requires sign-in with a Google account.

Optional Backup and Restore (Google Sign-In)

  • Account identifier: Google account identifier used for authentication.
  • Encrypted backup data: user-created collection data encrypted on-device before upload.

Advertising (Google AdMob)

The App uses Google AdMob to display advertisements. AdMob may automatically collect:

  • Device information
  • Advertising identifiers (e.g., Advertising ID)
  • Cookies or similar technologies

2. How We Use Information

  • To authenticate users for the optional backup/restore feature
  • To provide backup and restore of user data (encrypted)
  • To display ads and measure ad performance
  • To operate and improve the App

3. Data Storage and Encryption (Backup/Restore)

Backup data is encrypted on the user’s device using a password defined by the user before it is uploaded.

The App does not store, transmit, or have access to the user’s encryption password. If the password is lost, the encrypted backup data cannot be recovered or restored.

Only encrypted data is transmitted and stored. Backup storage is provided using Google services (e.g., Firebase/Firestore or equivalent backend storage).

4. Data Retention

Encrypted backup data is stored temporarily and is automatically deleted from the server after 7 days from the time of upload.

5. Data Sharing

The App does not sell user data.

  • Backup data: not shared with third parties. Stored only as encrypted data that cannot be decrypted by the App operator.
  • Ads: Ad-related data may be processed by Google AdMob in accordance with Google’s policies.

6. Security

Data transmitted to third-party services is sent securely using industry-standard encryption (e.g., HTTPS/TLS). Backup data is additionally protected by on-device encryption prior to upload.

7. Children’s Privacy

The App does not knowingly collect personal information from children. If you believe that a child has provided personal information, please contact us so that appropriate action can be taken.

8. Data Deletion

  • Encrypted backup data is automatically deleted after 7 days.
  • If the App provides an in-app option to delete backups, users can delete backups at any time.
  • Users may also request deletion or support via email: projectjhy@gmail.com

9. Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be posted on this page.

10. Contact

If you have any questions about this Privacy Policy, please contact:
projectjhy@gmail.com

Note: Backup and restore functionality is optional and disabled unless the user chooses to sign in and enable it.

1. 수집하는 정보

본 앱은 회원가입 없이 이용할 수 있습니다. 다만, 선택적으로 컬렉션 백업 및 복원 기능을 제공하며, 해당 기능을 사용하기 위해서는 Google 계정 로그인이 필요합니다.

선택적 백업 및 복원 (Google 로그인)

  • 계정 식별 정보: Google 로그인 인증을 위한 계정 식별 정보
  • 암호화된 백업 데이터: 사용자가 생성한 컬렉션 데이터(기기에서 암호화 후 업로드)

광고 (Google AdMob)

본 앱은 Google AdMob을 사용하여 광고를 표시합니다. AdMob은 광고 제공 및 성과 측정을 위해 다음 정보가 자동으로 수집될 수 있습니다.

  • 기기 정보
  • 광고 식별자(Advertising ID)
  • 쿠키 또는 유사 기술

2. 정보 이용 목적

  • 선택적 백업/복원 기능 제공을 위한 사용자 인증
  • 사용자 데이터 백업 및 복원 제공(암호화된 형태)
  • 광고 제공 및 광고 성과 측정
  • 앱 기능 운영 및 개선

3. 데이터 저장 및 암호화 (백업/복원)

백업 데이터는 사용자가 직접 설정한 비밀번호를 이용하여 기기 내에서 암호화된 후 서버로 업로드됩니다.

본 앱은 사용자의 비밀번호를 저장하거나 전송하지 않으며, 비밀번호를 알 수 없습니다. 비밀번호를 분실한 경우 암호화된 백업 데이터는 복원할 수 없습니다.

서버에는 복호화가 불가능한 암호화된 데이터만 저장됩니다. 백업 저장소는 Google 서비스 (예: Firebase/Firestore 또는 이에 준하는 백엔드 저장소)를 사용합니다.

4. 데이터 보관 기간

암호화된 백업 데이터는 임시로 저장되며, 업로드 시점으로부터 7일 후 자동으로 삭제됩니다.

5. 제3자 제공

본 앱은 사용자의 정보를 판매하지 않습니다.

  • 백업 데이터: 제3자에게 제공하지 않으며, 앱 운영자 또한 내용을 확인할 수 없습니다(암호화 저장).
  • 광고: 광고 관련 데이터는 Google AdMob 정책에 따라 처리될 수 있습니다.

6. 보안

제3자 서비스로 전송되는 데이터는 HTTPS/TLS 등 표준 보안 프로토콜을 통해 안전하게 전송됩니다. 백업 데이터는 업로드 전에 기기에서 추가로 암호화됩니다.

7. 아동의 개인정보 보호

본 앱은 아동으로부터 개인정보를 고의로 수집하지 않습니다. 아동의 개인정보 제공이 의심되는 경우 아래 연락처로 문의해 주시면 조치하겠습니다.

8. 데이터 삭제

  • 암호화된 백업 데이터는 업로드 후 7일이 지나면 자동 삭제됩니다.
  • 앱 내 백업 삭제 기능이 제공되는 경우 사용자는 언제든지 백업 데이터를 삭제할 수 있습니다.
  • 삭제/문의: projectjhy@gmail.com
